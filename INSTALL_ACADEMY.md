# Молодёжная академия + полный пакет всех правок

Это ПОЛНЫЙ архив — включает вообще всё, что менялось за всю сессию
(контракты, баги ЛЧ/травм, дизайн, свободные агенты, и теперь академия).
Распаковать поверх проекта, пути 1:1.

## Новая SQL-миграция

`supabase/migrations/0002_academy.sql` — накатить в Supabase SQL Editor
(если `0001_contracts_negotiations.sql` уже накатывал раньше — этот шаг
только для новой таблицы `academies` и `youth_prospects`, старую трогать
не нужно).

## Как устроена академия

Проспекты — **синтетические игроки**, не из CSV-датасета EA FC26.
Генерируются процедурно (позиция → архетип атрибутов → шум), с полным
набором характеристик, чтобы отрисовываться уже существующими
`PlayerModal`/`ExpandableStats` без единой правки в них.

- Каждый сезон академия выпускает 1-4 проспекта (16-18 лет), зависит от
  уровня академии (1-5).
- Уровень апгрейдится за деньги (`€800K → €1.8M → €3.5M → €6M`), списывается
  из бюджета клуба напрямую.
- **Повышение в первую команду** — создаёт контракт (роль «prospect»,
  зарплата ниже рыночной для их оверолла — юниоры играют дешевле) через уже
  существующую систему контрактов, и — ключевой момент — **`getPlayersByClub()`
  в `lib/players.ts` теперь подмешивает повышенных выпускников к обычному
  CSV-составу клуба**. Без этой правки повышение было бы бутафорией: контракт
  создавался бы, а игрок нигде не появлялся (ни в `/squad`, ни в лайнапе).
- Непроизведённых в тираж проспектов можно **отчислить** (просто пропадает
  из списка) или оставить — на следующий сезон они на год старше и чуть
  прибавляют к овероллу (в рамках потенциала); после 20 лет автоматически
  списываются, чтобы список не разрастался вечно.

## Где смотреть

Новый пункт меню **"Academy"** в сайдбаре (иконка — шапочка выпускника).
Показывает уровень академии, кнопку апгрейда, карточки выпускников этого
сезона с OVR/POT, кнопками "В команду" и отчислением. Клик по имени
игрока открывает полную карточку (тот же `PlayerModal`, что и в составе).

## Что проверить после установки

1. Накатить `0002_academy.sql`.
2. Открыть `/academy` на существующей карьере — академия и первый интейк
   должны создаться автоматически (лениво, как и с контрактами для старых
   карьер).
3. Повысить проспекта → открыть `/squad` → он должен появиться в общем
   списке игроков клуба.
4. Апгрейднуть академию → бюджет клуба должен уменьшиться на сумму
   апгрейда (проверить `/dashboard`, блок быстрых статов).
5. Пройти сезон целиком → новый интейк должен появиться в `/academy`
   автоматически на новый сезон.
